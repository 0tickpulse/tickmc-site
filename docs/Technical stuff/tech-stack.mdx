# Tech Stack

Note that this is not a complete list of all the software used by TickMC, but rather the main software used.

TickMC, the main Minecraft server, primarily uses the following technologies:

- [PaperMC](https://papermc.io/) - The server software
- [DenizenScript](https://denizenscript.com/) - The scripting engine used for the server
- [Bloom Host](https://bloom.host/) - The server hosting provider
- [Docker](https://www.docker.com/) - The containerization software used for the server and various other services
- [Pterodactyl](https://pterodactyl.io/) - The panel that Bloom.host uses

This website uses the following technologies:

- [React](https://reactjs.org/) - The frontend framework
- [TypeScript](https://www.typescriptlang.org/) - The programming language used for the frontend
- [TailwindCSS](https://tailwindcss.com/) - The CSS framework used for the frontend
- [Docusaurus](https://docusaurus.io/) - The static site generator
- [Vercel](https://vercel.com/) - The hosting provider

## Other services we provide

We have a [paste service](https://paste.tick-mc.net/), which is available at. It makes use of the following technologies:

- [Haste-server](https://github.com/toptal/haste-server) - The paste server software
- [Node.js](https://nodejs.org/) - The runtime for the paste server
- [Docker](https://www.docker.com/) - The containerization software used for the paste server
- [Bloom Host](https://bloom.host/) - The server hosting provider

## Other software we use

- [GitHub](https://github.com/0tickpulse) - Version control and issue tracking provider
- [Discord](https://discord.com/) - Main communication platform
- [Caddy](https://caddyserver.com/) - A backend webserver used for various services
- [Cloudflare](https://cloudflare.com/) - The primary DNS provider. Note that we use this in conjunction with Bloom's reverse proxy.
